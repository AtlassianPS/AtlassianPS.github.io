Contribute

This gallery can only be useful with as much input/collaboration as possible. So please consider submitting your ideas / code.

To submit your code, you need to fork this repository, change this file and submit your changes as a Pull Request.
You can find a detailed instruction here - link broken for now.

  1. Fork it ( https://github.com/AtlassianPS/AtlassianPS.github.io/fork )
  2. Clone your fork (git clone https://github.com/<YOU>/AtlassianPS.github.io)
  3. Create your feature branch (git checkout -b my-new-feature)
  4. Commit your changes (git commit -am 'Add some feature')
  5. Push to the branch (git push origin my-new-feature)
  6. Create new Pull Request
